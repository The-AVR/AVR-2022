![](nano_combo.png)

- Unbox your Jetson
- Unbox WiFi Adapter
- Unscrew Jetson top
- Push the tabs out

![](annotated_jetson.png)

- Pull out the Jetson Nano Module
- Pull black screw

![](wifi_card_annotes.png)

- Slide in the WiFi card
- Put in black screw
- Slide module back into place
- Replace screws

![](DSC02233.jpg)

{{% alert title="Success" color="success" %}}
Now you're ready to flash the SD Card!
{{% /alert %}}
